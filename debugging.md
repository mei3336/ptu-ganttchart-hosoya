# Skill: デバッグ

## いつ使うか
エラーが出た、または期待した動作にならないとき

## AIへの依頼テンプレート
```
以下のエラーが出た。原因と修正方針を、エンジニア初心者向けに
「なぜこのエラーが起きるのか」から説明して。
修正コードだけでなく、修正理由も日本語で書いて。

【エラーメッセージ】
（貼る）

【関連コード】
（貼る）

【何をしようとしていたか】
（例：タスクを削除しようとしたら発生した、など）
```

## 確認すること（レビューポイント）
- 「なぜエラーが起きたか」の説明が、コードの該当箇所と対応しているか
- 同じ種類のミスが他の箇所にもないか、AIに確認してもらう
- 修正後、同じ手順で再現しないか動作確認する

## 記録すること
解決したエラーは一言メモを残す（同じ原因で何度もハマらないようにするため）
