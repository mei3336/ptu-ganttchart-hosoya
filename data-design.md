# Skill: データモデル設計

## いつ使うか
新しく保存したいデータが出てきたとき（例：「タスクにタグをつけたい」など）

## AIへの依頼テンプレート
```
IndexedDBで〇〇を保存したい。
保存したい項目：（例：タスク名・完了状態・作成日時）
やりたい絞り込み・並び替え：（例：完了/未完了で絞り込みたい）

この情報から、オブジェクトストア構成・keyPath・必要なインデックスを
理由つきで提案して。コードはまだ書かないで。
```

## 確認すること（レビューポイント）
- 提案されたストア構成が「なぜそうなるのか」の理由が説明されているか
- keyPath（一意に識別するためのキー）が妥当か
- インデックスが「絞り込みたい項目」に対応しているか
- CLAUDE.mdの命名規約（明確な名前）に沿っているか
